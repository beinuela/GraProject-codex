# 湛江科技学院理工类本科论文格式提要

This reference is extracted from the school sample document `1-2.湛江科技学院本科毕业论文（设计）撰写简要参考格式示例-理工类.doc`.

When the sample document and this summary disagree, visually match the sample document first.

## 1. Overall Ordering

1. Cover page
2. Originality statement
3. Copyright authorization
4. Chinese abstract page
5. English abstract page
6. Automatic table of contents
7. Main body
8. References
9. Acknowledgements
10. Appendices (optional)

## 2. Cover Page

- Top line centered: `2026 届本科生毕业论文（设计）`
- Chinese title:
  - Xiao-er
  - bold
  - black
  - centered
  - single spacing
- English title:
  - Times New Roman
  - Xiao-er
  - bold
  - centered
  - title case
- Student info block:
  - centered
  - Chinese in Songti
  - digits / Latin in Times New Roman
  - Xiao-san

## 3. Abstract Pages

### Chinese Abstract Page

- Thesis title: San-hao, bold, black, centered, new page
- `摘 要：`:
  - first-line indent 2 chars
  - bold
  - Xiao-si
- Abstract body:
  - Songti Xiao-si
  - justified
  - minimum 22 pt line spacing
  - 300-600 Chinese characters recommended
- `关键词：`:
  - first-line indent 2 chars
  - bold
  - Xiao-si
- Keywords separated by half-width `; `

### English Abstract Page

- English title:
  - Times New Roman
  - bold
  - San-hao
  - centered
  - new page
- `Abstract:`:
  - Times New Roman
  - bold
  - Xiao-si
- Body:
  - Times New Roman
  - Xiao-si
  - justified
  - 1.5 line spacing
- `Keywords:`:
  - Times New Roman
  - bold
  - Xiao-si
- Keywords:
  - Times New Roman Xiao-si
  - 1.25 line spacing
  - lowercase except proper nouns

## 4. TOC

- New page
- `目 录` centered, San-hao, bold, black
- Use automatic Word TOC only
- Show levels 1-3

## 5. Main Body Typography

### Level 1 Heading

- centered
- black, bold, San-hao
- paragraph before/after 0.5 line
- minimum 22 pt line spacing
- new page

### Level 2 Heading

- left aligned
- black, bold, Si-hao
- paragraph before/after 0.5 line
- minimum 22 pt line spacing

### Level 3 Heading

- left aligned
- black, bold, Xiao-si
- paragraph before/after 0.5 line
- minimum 22 pt line spacing

### Body Text

- Chinese in Songti
- Latin letters and Arabic digits in Times New Roman
- Xiao-si
- first-line indent 2 chars
- justified
- minimum 22 pt line spacing

## 6. Figures

- Caption below figure
- `图X-X` numbering by chapter
- Caption centered, Songti, Xiao-si, bold
- Keep figures simple and engineering-oriented

## 7. Tables

- Caption above table
- `表X-X` numbering by chapter
- Caption centered, Songti, Xiao-si, bold
- Prefer three-line tables
- Use continuation marker `（续）` for multi-page tables
- Source notes may appear below tables

## 8. Equations

- Equation centered on its own line
- Number right aligned, format `（X-X）`

## 9. References

- New page
- Title `参考文献` centered, San-hao, bold, black
- Sequential numbering in square brackets
- Chinese in Songti, Latin/digits in Times New Roman
- Wu-hao
- 1.5 line spacing
- hanging indent 2 chars

## 10. Acknowledgements And Appendices

- `致 谢` on a new page, centered, San-hao, bold, black
- `附录A/附录B` on a new page, centered, San-hao, bold, black
- Appendix level 2 headings use Si-hao black bold
- Appendix level 3 headings use Xiao-si black bold

## 11. Delivery Hygiene

- Delete instructional text, notes, placeholder boxes, and any sample-only hints.
- Keep all visible text black.
- Remove hyperlink styling from headings and body text.
