# 湛江科技学院理工类本科论文格式提要

This reference is extracted from the school sample document `1-2.湛江科技学院本科毕业论文（设计）撰写简要参考格式示例-理工类.doc`.

When the sample document and this summary disagree, visually match the sample document first.

## 1. Overall Ordering

The sample shows this front-to-back order:

1. Cover page
2. Originality statement
3. Copyright authorization
4. Chinese title / Chinese abstract / Chinese keywords
5. English title / English abstract / English keywords
6. Automatic table of contents
7. Main body
8. References
9. Acknowledgements
10. Appendices (optional)

Every major front-matter section starts on a new page.

## 2. Cover Page

- Top line: `2026 届本科生毕业论文（设计）`, centered.
- Chinese title: Xiao-er size, bold, black, centered, single spacing.
- English title: Xiao-er size, Times New Roman, bold, centered, title case.
- Student info block:
  - centered
  - Chinese text in Songti
  - Arabic digits / Latin text in Times New Roman
  - Xiao-san size
- Keep the same label order as the sample:
  - 学生姓名
  - 学院
  - 行政班级
  - 学号
  - 指导教师
  - 完成时间

## 3. Originality Statement / Authorization Pages

- Each starts on a new page.
- Title centered, black, bold, San-hao.
- Body text:
  - Chinese in Songti
  - Latin text / digits in Times New Roman
  - Xiao-si
  - justified
  - first-line indent 2 chars where applicable
- Signature lines must remain as formal signature/date placeholders, not explanatory notes.
- Remove any template text box notes such as "阅读后删除此文本框".

## 4. Chinese Abstract Page

- Thesis Chinese title:
  - black, bold, San-hao
  - centered
  - start on a new page
- If title is one line:
  - paragraph before: 0.5 line
  - paragraph after: 1 line
- If title spans multiple lines:
  - first line before: 0.5 line
  - middle lines before/after: 0
  - last line after: 1 line
- Title line spacing: minimum 22 pt

### Abstract

- Label `摘 要：`
  - first-line indent 2 chars
  - black, bold, Xiao-si
- Abstract body:
  - Songti Xiao-si
  - justified
  - minimum 22 pt line spacing
  - no figures, tables, notes, formulas, unexplained abbreviations
  - 300-600 Chinese characters recommended

### Keywords

- Label `关键词：`
  - first-line indent 2 chars
  - black, bold, Xiao-si
- Content:
  - Songti Xiao-si
  - minimum 22 pt line spacing
  - separated with half-width `; `
  - usually 3-5 keywords, no more than 8

## 5. English Abstract Page

- English thesis title:
  - Times New Roman
  - bold
  - San-hao
  - centered
  - title case
  - start on a new page
- `Abstract:` label:
  - Times New Roman
  - bold
  - Xiao-si
- English abstract body:
  - Times New Roman
  - Xiao-si
  - justified
  - 1.5 line spacing
- `Keywords:` label:
  - Times New Roman
  - bold
  - Xiao-si
- Keywords:
  - Times New Roman Xiao-si
  - 1.25 line spacing
  - lowercase except proper nouns

## 6. Table Of Contents

- Starts on a new page.
- Title `目 录`:
  - centered
  - black, bold, San-hao
  - paragraph before about 1.5 lines
  - paragraph after about 1 line
- Must be an automatic Word table of contents, never hand-typed.
- The sample shows TOC levels 1-3.

### TOC Entry Styles Shown In Sample

- Level 1 entry:
  - left aligned
  - black, bold, Xiao-si
  - minimum 22 pt line spacing
- Level 2 entry:
  - left aligned
  - Songti Wu-hao
  - left indent 1 char
  - minimum 22 pt line spacing
- Level 3 entry:
  - left aligned
  - Songti Wu-hao
  - left indent 2 chars
  - minimum 22 pt line spacing

## 7. Main Body

### Chapter Title (Level 1)

- Example: `1 引言`
- centered
- black, bold, San-hao
- paragraph before 0.5 line
- paragraph after 0.5 line
- minimum 22 pt line spacing
- starts on a new page

### Section Title (Level 2)

- left aligned
- black, bold, Si-hao
- paragraph before 0.5 line
- paragraph after 0.5 line
- minimum 22 pt line spacing

### Subsection Title (Level 3)

- left aligned
- black, bold, Xiao-si
- paragraph before 0.5 line
- paragraph after 0.5 line
- minimum 22 pt line spacing

### Level 4 And Below

- normally inline or paragraph-level titles such as `（1）...`
- first-line indent 2 chars
- Songti Xiao-si
- paragraph before/after 0.5 line
- minimum 22 pt line spacing

### Body Text

- Chinese: Songti
- Latin letters and Arabic numbers: Times New Roman
- Xiao-si
- first-line indent 2 chars
- justified
- minimum 22 pt line spacing

## 8. Figures

- Use continuous Arabic numbering by chapter, e.g. `图3-1`.
- Even if there is only one figure, still use `图1-1` style numbering.
- Caption goes below the figure.
- Caption format:
  - Songti
  - Xiao-si
  - bold
  - centered
- Figure content must be concise and match the text.

## 9. Tables

- Use continuous Arabic numbering by chapter, e.g. `表3-1`.
- Table caption goes above the table.
- Caption format:
  - Songti
  - Xiao-si
  - bold
  - centered
- No punctuation in the table title, and no trailing punctuation.
- Prefer three-line tables.
- Multi-page tables repeat the title and add `（续）`.
- Show units in headers or near the table header as needed.
- Source note can be placed below the table.
- Sample note format for source:
  - KaiTi
  - Wu-hao
  - single spacing
  - first-line indent 2 chars

## 10. Equations And Numeric Writing

- Equations use continuous Arabic numbering by chapter, e.g. `（3-1）`.
- Equation body centered on its own line.
- Equation number right aligned.
- Formula and number do not need dotted leaders.
- Use Arabic numerals for exact dates, times, quantities, and counts.
- Use `～` for numeric/time ranges, except year-to-year ranges may use `-`.
- Use spaces for thousand grouping instead of commas when required by the sample's rule note.

## 11. References

- Starts on a new page.
- Title `参考文献`:
  - centered
  - black, bold, San-hao
  - paragraph before/after 0.5 line
  - minimum 22 pt line spacing
- Use sequential citation numbering in square brackets, ordered by first appearance in the text.
- Body formatting:
  - Chinese in Songti
  - Latin letters/digits in Times New Roman
  - Wu-hao
  - 1.5 line spacing
  - hanging indent 2 chars
- The sample notes a baseline of at least 10 references and at least 2 foreign-language references; if the department provides a stricter rule, follow the department rule.
- Common reference type markers:
  - `M` book
  - `C` conference / proceedings
  - `G` collection
  - `N` newspaper
  - `J` journal
  - `D` thesis
  - `R` report
  - `S` standard
  - `P` patent
  - electronic types `DB`, `CP`, `EB`, etc.

## 12. Acknowledgements

- Starts on a new page.
- Title `致 谢`:
  - centered
  - black, bold, San-hao
  - paragraph before/after 0.5 line
  - minimum 22 pt line spacing
- Body text follows normal body formatting.

## 13. Appendices

- Optional.
- Each appendix starts on a new page.
- Appendix title such as `附录A` or `附录B`:
  - centered
  - black, bold, San-hao
  - paragraph before/after 0.5 line
  - minimum 22 pt line spacing
- Appendix level 2 headings such as `A.1`:
  - left aligned
  - black, bold, Si-hao
  - paragraph before/after 0.5 line
- Appendix level 3 headings such as `A.1.1`:
  - left aligned
  - black, bold, Xiao-si
  - paragraph before/after 0.5 line
- Appendix body follows normal body formatting.

## 14. Page Numbers And Sections

- The sample visibly shows centered Arabic page numbers on main-text pages.
- In Word, use separate sections so front matter and main text can be controlled independently.
- If the sample or department template shows suppressed page numbers on front matter, keep them hidden there and start visible numbering from the main text section.

## 15. Hygiene Rules For Final Delivery

- Delete all placeholder instructions, template hints, colored notes, and text boxes used only for explanation.
- Ensure all visible thesis text is black unless a school requirement explicitly says otherwise.
- Keep headings free of hyperlink styling.
